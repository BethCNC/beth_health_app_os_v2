# Design System Spec

## Design Direction
Calm clinical clarity with low cognitive load and explicit information hierarchy.

## Token Categories
- Color: high-contrast neutrals + restrained accent for actions.
- Spacing: 4/8-based scale with generous panel padding.
- Typography: accessible sans stack with clear heading rhythm.
- Elevation: minimal layered shadows to separate content without visual noise.

## Primitive Components
- `Card`
- `StatusBadge`
- Shell navigation primitives
- Data tables
- Timeline cards

## Density Modes
- Summary mode default.
- Detailed mode toggle for advanced review.

## Clinician View Requirements
- Snapshot first.
- Timeline and documents as adjacent drill-down tabs/sections.
- Citation visibility at point-of-claim.
