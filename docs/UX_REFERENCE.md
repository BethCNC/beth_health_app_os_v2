# UX Reference

## Epic-Inspired Layout
- Left navigation for primary modules
- Patient context header with demographics and status
- Workspace tabs: Summary, Timeline, Labs, Medications, Documents

## Interaction Notes
- Context stays visible while switching tabs
- Tabs are horizontal and persist across navigation
- Summary shows high-level, scannable cards

## Visual Direction
- Dense but readable information layout
- Clear typographic hierarchy
- Subtle dividers to separate sections
